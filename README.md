# Cypress JavaScript End-to-End Testing

This repository contains **end-to-end (E2E) tests** implemented in **Cypress** using JavaScript. 

## Tools 
- Cypress
- JavaScript (ES6)
- Node.js

## Web Application Under Test

All tests are performed on the publicly available demo site:

**[The Internet - Herokuapp](https://the-internet.herokuapp.com)**

## Features Tested

- JavaScript Alert Handling  
- JavaScript Confirm Alerts 
- JavaScript Prompt Alerts
